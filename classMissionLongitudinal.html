Detailed Description

template<typename T>
class MissionLongitudinal< T >

This is the class responsible for representing the longitudinal missions. The class MissionLongitudinal is a child class of the class Mission. These classes are implemented as templates, which means that single or double precisions may be used to represent real numbers. This class' objective is to guide each aircraft to implement the longitudinal mission. Following this line of reasoning, this class takes into account its private attributes as well as the attributes of the class MissionSolver with the objective of executing the proposed mission.

vector< vector< T > > MissionLongitudinal< T >::run_mission ( Mission< T > *  origem,
MissionSolver< T > *  ms,
vector< vector< T > > &  ref 
)
virtual

This is one of the most important methods of this class. It is responsible for making each aircraft to execute the proposed mission. With this objective in mind, it takes into account an object of the class Mission, which is an object of the previous mission executed by the MissionSolver. Although its body only returns an empty vector of references, this method is important to keep compatibility with the other children classes of the class Mission. Besides, it is still possible to execute longitudinal missions because all necessary parameters and methods are in the class Mission and in the class MissionSolver. Since the radius and angles of the circular maneuvers and length of the straight forward maneuvers are all known, each aerial vehicle's trajectory can therefore be deterministically calculated for all missions.

Parameters
origemThe pointer to an object of the class Mission, which represents an object of the previous mission.
msThe pointer to an object of the class MissionSolver. This object stores all other attributes necessary to the implementation of this mission.
refIs an attribute passed as a reference (it is not altered by this method).
Returns
An empty vector.
