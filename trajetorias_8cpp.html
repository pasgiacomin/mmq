◆ adaptacao_indice_lat_long()

int adaptacao_indice_lat_long ( int  indice,
int  n_aeronaves 
)

This function calculates the new order of a given aircraft in the squadron if a latitudinal mission would be executed. The order of an aircraft is the same as the index of the same aircraft if zero latitudinal missions were executed. The aircraft's order after executing several latitudinal missions is determined by calling this function several times, i.e., the number of calls is the same as the number of latitudinal missions.

Parameters
indiceThe initial order of the aircraft before executing the latitudinal mission.
n_aeronavesThe number of aerial vehicles in the squadron.
Returns
The new order of the aircraft in the squadron, if a latitudinal mission would be executed.

This function is equivalent to the function "trajetoria_em_frente_3D", but it has one important difference: it does not produce the references and the vector of time, but it only calculates the new distance of the aircraft after executing the proposed maneuver. This is necessary for real-time prediction purposes.

Parameters
x0Is the longitudinal coordinate of the 3D source point.
y0Is the latitudinal coordinate of the 3D source point.
h0Is the height of the 3D source point.
phiIs the aircraft's rotation angle with regard to the vertical axis.
betaIs the aircraft's rotation angle with regard to the latitudinal axis.
distIs the distance flown by the given aircraft.
distanciaIs the distance calculated as a result from several numeric sums during the maneuver (one sum for each time step). This distance is different from the dist argument, which is a specification argument.
Returns
This function returns the new position of the aircraft after executing the proposed maneuver.
