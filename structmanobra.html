Detailed Description

template<typename T>
struct manobra< T >

This data structure is used to generically represent the two most basic maneuvers, i.e., the "to turn" maneuver and the "to go forward" maneuver. All of them begin at a specific point, which is represented here by the variables x0, y0 and h0. The angles phi and beta are used by both maneuvers, while the angles theta and alpha are only used by the "to turn" maneuver. The flag "plotar" was used in the beginning of the development phase, where it was necessary to closely observe how each maneuver have been generated. However, it is currently no more necessary. The parameter id was created due to the same reason. The function pointer called traj is able to generically represent both of the maneuvers.

When a maneuver is specified, it is necessary to say its velocity and its distance. These parameters are represented by the variables vel and dist, respectively.
