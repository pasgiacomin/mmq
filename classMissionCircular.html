Detailed Description

template<typename T>
class MissionCircular< T >

This is the class responsible for representing the circular missions. The class MissionCircular is a child class of the class Mission. These classes are implemented as templates, which means that single or double precisions may be used to represent real numbers. This class' objective is to guide each aircraft to implement the circular mission. Following this line of reasoning, this class takes into account its private attributes as well as the attributes of the class MissionSolver with the objective of executing the proposed mission.

template<typename T >
vector< vector< T > > MissionCircular< T >::run_mission ( Mission< T > *  anterior,
MissionSolver< T > *  ms,
vector< vector< T > > &  ref 
)
virtual

This is the most important method of this class. It is responsible for making each aircraft to execute the proposed mission. With this objective in mind, it takes into account an object of the class Mission, which is an object of the previous mission executed before it. The object of the class MissionSolver has all other attributes necessary to the execution of this mission. As a result, the proposed method produces a sequence of 3D points responsible for guiding each aircraft during the execution of the proposed mission.
Since the radius and angles of the circular maneuvers and length of the straight forward maneuvers are all known, each aerial vehicle's trajectory can therefore be deterministically calculated for all missions.

Parameters
anteriorThe pointer to an object of the class Mission, which is related to the previous mission.
msThe pointer to an object of the class MissionSolver. This object stores all other attributes necessary to the implementation of this mission.
refIs an attribute passed as a reference (it is altered by this method), which will store the sequence of 3D points that are used as reference commands to guide each aircraft in the execution of this mission.
Returns
The content of the variable ref, as described above.

Implements Mission< T >.
